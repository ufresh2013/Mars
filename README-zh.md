## 📦 安装

```bash
# node版本 v22.11.0
# 安装依赖
yarn

# 本地启动
yarn dev

# 打包并发布到github page
yarn d
```

## 📝 选型

Mars基于以下包进行构建：

- Vite
- React
- ReactFlow
- React Remark
- bytemd
- Antv G2plot
