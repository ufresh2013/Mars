# Mars
Mars 是一款强大的工具，它能够从类 Markdown 文本生成 Word 文档、PPT、流程图和图表数据报告。
 
- Playground: https://ufresh2013.github.io/Mars/

## 特性
- **Markdown 支持**：使用熟悉的 Markdown 语法进行内容创作。
- **多格式输出**：直接转化为 Word 文档、PPT、流程图和图表。

## 命令
```bash
# node版本 v22.11.0
# 安装依赖
yarn

# 本地启动
yarn dev 

# 打包并发布到github page 
yarn d
```
